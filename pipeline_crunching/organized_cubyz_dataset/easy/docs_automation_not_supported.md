# Machines and Automation

Cubyz does not currently have any machine or automation systems. There are no conveyor belts, auto-crafters, factories, or redstone-like circuitry of any kind.

All crafting in Cubyz is manual. Items are crafted either through the Crafting Menu, which automatically lists every possible craftable item, or through the Workbench, where material items are combined by hand to produce items with different speed/damage or durability/slow properties.

There is currently no way to automate resource gathering, item transport, or crafting in Cubyz.
