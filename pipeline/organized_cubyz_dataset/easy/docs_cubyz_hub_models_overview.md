# Cubyz Hub Models

Cubyz Hub Models is a separate service from the main "Cubyz Hub" addon marketplace (addons.ashframe.net), also built by iNiKKo under the Ashframe umbrella. Where the addon marketplace shares full Cubyz addons, Cubyz Hub Models is specifically for sharing 3D models and custom skins for existing in-game entities.

Users can upload a `.glb` 3D model and a texture image for either a full custom model or a skin-only reskin of an existing entity (for example, a custom skin for `cubyz:snale`). Uploaded assets can be browsed, sorted (newest, oldest, most voted), and voted on by other users.

This is a distinct tool from the addon marketplace -- it does not host full addons, only standalone models/skins that reference an existing in-game entity by its ID.
