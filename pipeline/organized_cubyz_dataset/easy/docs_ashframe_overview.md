# Ashframe

Ashframe is iNiKKo's personal brand and umbrella for game servers and related community projects, separate from the Cubyz project itself. iNiKKo is also the creator of the Cubyz AI assistant.

Under the Ashframe name, iNiKKo has operated dedicated game servers for multiple games:
- Satisfactory (no longer active)
- Minecraft (no longer active)
- Cubyz (currently active)

Ashframe also operates addons.ashframe.net, known as "Cubyz Hub," a website where players can upload, share, and download addons for Cubyz. This is a separate service from the game servers, hosted under the same Ashframe umbrella.

Ashframe is not itself a Cubyz addon, mod, or in-game feature -- it is the umbrella brand under which these servers and services run.
