## Summary
CachedFractalNoise implements a spatially cached fractal noise generator for terrain heightmaps, using a two‑dimensional array cache with voxel‑aligned coordinates and lazy region generation to avoid full initialization.

## Explanation
The struct CachedFractalNoise holds a MapFragmentPosition (wx, wy, voxelSize, voxelSizeShift) and an Array2D(f32) cache. init allocates the cache via main.globalAllocator, sets its dimensions to maxSize+1 where maxSize = size/voxelSize, and fills corner entries with 0 using @memset; it also computes reducedScale = scale/voxelSize and iterates over a coarse grid (step = reducedScale) to seed those corners. deinit calls cache.deinit on the global allocator. getRandomValue uses main.random.initSeed2D seeded by worldSeed plus the given wx, wy, then returns nextFloat minus 0.5. getGridValue maps a relative coordinate into world space by adding self.pos.wx and self.pos.wy scaled by voxelSize, then delegates to getRandomValue. generateRegion is called lazily; it aligns x and y to the nearest voxel boundary via bitwise AND with ~ (voxelSize‑1), ensures higher points are generated by calling _getValue on the top‑right corner of the region, computes midpoints xMid and yMid, and then averages neighboring cache entries along each axis before adding a random factor derived from voxelSize. The center point is averaged over its four neighbors plus the same random factor. _getValue checks if cache.get(x,y) != 0; if so it returns that value, otherwise it computes minShift = @min(@ctz(x), @ctz(y)) and calls generateRegion with a size of (2 << minShift). getValue converts world coordinates to cache indices by subtracting the fragment offset, right‑shifting by voxelSizeShift, then calls _getValue and finally subtracts self.scale as a float. All functions are pub except getGridValue and generateRegion which are internal helpers.
