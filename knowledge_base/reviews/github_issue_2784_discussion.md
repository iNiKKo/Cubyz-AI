## Explanation
Discussion about adding an option to show both local and external IP addresses in Cubyz to improve usability for LAN gaming. The main concern is making the concept of IP addresses more accessible to non-technical users, as currently entering the public IP address on a local network causes confusion. Suggestions include adding explanatory text or a toggle option that clearly indicates which IP address to use for LAN versus online play. This approach would be similar to Minecraft's feature where invitations only work on LAN networks. The proposed solution involves putting clear instructions in the user interface, such as telling users to use one IP if they are inviting someone not on their local network and another if they are inviting someone locally.

- What specific text should be added to clarify which IP address to use in different scenarios?
- Can we implement a toggle option that shows both IP addresses clearly?
