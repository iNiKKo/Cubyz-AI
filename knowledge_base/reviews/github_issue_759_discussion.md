## Summary
The issue proposes displaying each crafting recipe as a single icon with ingredients shown in tooltips, allowing multiple recipes to fit into a single row for a more compact inventory window.

## Explanation
The discussion suggests that recipes with the same result should be displayed as a single item. Right-clicking on this item would allow users to choose specific ingredients, while left-clicking could trigger a default action. This change aims to optimize the user interface by reducing clutter and improving usability.
