## Summary
The issue proposes adding a list of unique materials used in tool tooltips to aid in reproducing recipes.

## Explanation
The discussion revolves around enhancing the tooltip functionality by including detailed material lists. The maintainer suggests creating an additional table similar to another feature (#1071) that displays the original recipe, which could also be beneficial for analyzing tools obtained through loot. The primary concern is avoiding clutter in the standard tooltip, hence the proposal of a hotkey to toggle advanced tooltips with full material information.
