## Summary
The UI using block textures is considered too noisy, affecting readability. The maintainer suggests making the slots opaque to improve clarity.

## Explanation
The issue highlights that the current UI design with detailed block textures is overwhelming and distracting, affecting readability. The maintainer suggests a simple solution of making the item slots opaque to reduce visual clutter without losing the concept of using real materials in the UI. This change aims to enhance usability while maintaining aesthetic appeal. Specifically, the textures for the item slots are located in `assets/cubyz/ui/inventory`. Additionally, the user has indicated that they can update these textures themselves if necessary.
