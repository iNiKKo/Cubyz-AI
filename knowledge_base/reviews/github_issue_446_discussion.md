## Summary
The maintainer declines to add all 6000 minerals to the base game or maintain mods, suggesting users create their own repositories for such extensive modding.

## Explanation
The maintainer of Cubyz has decided not to include a comprehensive list of over 6000 minerals in the base game due to the significant maintenance burden. Instead, they propose that interested developers create their own separate repositories for mods containing these minerals and their crafting recipes. The maintainer also suggests maintaining a curated list of known addons/mods only if there is at least one addon with new content, indicating a desire to avoid overextending the scope of the base game without substantial community contribution.
