## Summary
Discussion about rendering issues for glass items in Cubyz version 0.0.1.

## Explanation
The issue revolves around the visibility of glass items within the game environment. Users report that while some transparent blocks like ice and amber are visible, glass items appear invisible both when held and as item drops. The maintainer suggests exploring alternative rendering methods since transparency and dithering are not viable solutions. Users propose making glass opaque or using a fallback texture for better visibility.
