## Summary
Discussion about rendering issues for glass items in Cubyz version 0.0.1.

## Explanation
Discussion about rendering issues for glass items in Cubyz version 0.0.1. Users report that while some transparent blocks like ice and amber are visible, glass items appear invisible both when held and as item drops. The steps to reproduce the issue include selecting a cyan glass item and observing its invisibility despite having view model on. Transparency and dithering are not viable solutions for rendering glass items due to absorption effects. Users propose making glass opaque or using a fallback texture for better visibility.
