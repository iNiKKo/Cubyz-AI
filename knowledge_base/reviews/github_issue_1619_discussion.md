## Summary
The proposal suggests hiding the cursor in textboxes when they are not focused, aligning with standard OS behavior. The debate focuses on whether to also remove selections when focusing away from a textbox.

## Explanation
The issue discusses the current behavior of textboxes in Cubyz where cursors do not disappear even when the textbox loses focus. The proposal aims to improve user experience by making the cursor visible only when the textbox is active, similar to standard OS practices. The reviewer emphasizes simplicity and argues against emulating complex behaviors seen in other applications, suggesting that Cubyz should maintain a single cursor across the entire application for clarity.
