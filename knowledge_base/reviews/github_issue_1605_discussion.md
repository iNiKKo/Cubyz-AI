## Explanation
The discussion revolves around the idea of having exclusive crafting stations for each tool type, with pros and cons highlighted regarding immersion, UI clutter, and design consistency. The issue proposes moving the tool selection button into the world by creating separate crafting stations for each tool type. This change aims to enhance immersion and provide more decoration options but introduces challenges such as increased movement between stations and potential UI clutter. Each crafting station should have visibly distinct textures including shape of the tool both on top and on the side, so players don't have to remember details of the station. The maintainers suggest a generic UI for new tools and distinct textures for each station to improve player experience. For example, the pickaxe table would be crafted from wood since it's the only thing accessible at that stage in progression, while other stations could use up to copper. There is also discussion about extending this concept to armor (3 slots) and accessories (around 10 types), with concerns raised about excessive variety leading to a cluttered design similar to modded Minecraft. The team ultimately decides to proceed with separate tables for different classes of items but not for every individual type, focusing on progression-based unlocking. For instance, bows would have their own table, and all kinds of rings could share one table.
