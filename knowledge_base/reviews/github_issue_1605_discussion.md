## Summary
The discussion revolves around the idea of having exclusive crafting stations for each tool type, with pros and cons highlighted regarding immersion, UI clutter, and design consistency.

## Explanation
The issue proposes moving the tool selection button into the world by creating separate crafting stations for each tool type. This change aims to enhance immersion and provide more decoration options but introduces challenges such as increased movement between stations and potential UI clutter. The maintainers suggest a generic UI for new tools and distinct textures for each station to improve player experience. There is also discussion about extending this concept to armor and accessories, with concerns raised about excessive variety leading to a cluttered design similar to modded Minecraft. The team ultimately decides to proceed with separate tables for different classes of items but not for every individual type, focusing on progression-based unlocking.
