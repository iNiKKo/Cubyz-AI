## Summary
The Creative menu cannot be closed using the same 'C' input that opens it, unlike other menus. The discussion suggests adding a visual button inside the inventory to open the Creative menu and potentially removing or modifying the keyboard shortcut.

## Explanation
The issue revolves around inconsistent behavior in the Creative menu where pressing 'C' does not close the menu when already open, unlike other menus that use the same key for both opening and closing. The discussion explores potential solutions, including adding a visual button inside the inventory to access the Creative menu. This approach aims to improve user experience by making the functionality more intuitive and discoverable, especially for new players who may not be aware of the 'C' shortcut. Additionally, there is a suggestion to limit keyboard shortcuts to enhance controller support.
