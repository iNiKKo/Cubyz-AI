## Explanation
The issue discusses changing the function of the Escape key to open a different menu instead of opening the inventory. The current behavior is that pressing Escape opens the inventory, but there's a proposal to change it so that Escape opens a menu similar to what the hamburger button does. This menu includes additional options beyond just the inventory. The maintainer initially considered removing the extra keybind if the inventory is always open, but later preferred keeping the Escape key for accessing the main menu, more like in Terraria where pressing only one keybind (Escape) puts you into a comprehensive menu.
