## Summary
The maintainer has decided against adding Android support to Cubyz due to the game's current state and the complexity involved in maintaining multiple platforms.

## Explanation
The maintainer explains that Cubyz is not yet a fully developed game, and there are still many features missing. Additionally, the maintainer is currently facing issues with Java, particularly with garbage collector-induced lag spikes, which might lead to a language switch. Supporting Android would complicate this process. The maintainer also highlights the ongoing maintenance burden of ensuring compatibility across different platforms and adapting new graphics features for both OpenGL and OpenGL ES. Despite these challenges, the open-source nature of Cubyz allows for third-party developers to create an Android version independently.
