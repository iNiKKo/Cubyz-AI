## Summary
The inventory crafting menu now displays a short text when there are no ingredients instead of showing a degenerate window.

## Explanation
The change addresses the issue where opening the inventory crafting menu without any ingredients resulted in an undesirable visual state (degenerate window). The update ensures that players receive a clear and informative message, improving user experience by providing context about the lack of available ingredients. This modification aligns with the goal of enhancing usability and reducing confusion for users.
