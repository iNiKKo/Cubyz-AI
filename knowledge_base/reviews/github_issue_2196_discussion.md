## Summary
Discussion about the shading issue with block contrast in Cubyz, noting it as a temporary workaround and suggesting removal once proper lighting is implemented.

## Explanation
The issue revolves around the shading of grass blocks when block contrast is enabled. The maintainer confirms that this is a side effect of another change (#2072) and considers it a temporary issue since the default setting does not exhibit this problem. The feature 'block contrast' is described as a workaround until proper sun/moonlight and shadows are implemented, at which point the maintainer believes the issues will be less noticeable due to overall darker shading on both the backside of grass and the ground.
