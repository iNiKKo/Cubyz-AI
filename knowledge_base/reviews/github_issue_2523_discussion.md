## Summary
The 'grey' background for clicked slots is removed due to inconsistent user experience.

## Explanation
The maintainer decided to remove the 'grey' background that appears when a slot is clicked and the mouse is moved around. The original intention was to match the on-press display of buttons, but it was found to be inconsistent and felt weird overall. Buttons have fewer behavior modes compared to slots, making this visual feedback less necessary.
