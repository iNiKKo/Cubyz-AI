## Summary
The issue proposes adding a popup notification before downloading addon data from the server, addressing player concerns about viruses or illegal images. The discussion suggests that this popup could also serve as a download progress indicator.

## Explanation
The issue proposes adding a popup notification before downloading addon data from the server to address player concerns about viruses or illegal images. The proposed solution involves displaying a popup that informs players of the impending download and provides options to either proceed or cancel, as well as opening the respective folder in the explorer. Additionally, there's a suggestion to extend this popup to include a scrollable list of all content to be downloaded and individual file progress indicators, enhancing user experience by providing more detailed information about the download process.
