## Explanation
The discussion revolves around refining the procedural crafting system in Cubyz. The maintainers propose separating tool attributes into 'Power' and 'Hardness' to better differentiate between a tool's ability to damage blocks (Power) and its resistance to being damaged (Hardness). They suggest using a dropdown menu during tool creation to select the desired tool type, which would then apply specific formulas for determining stats. The implementation of #594 is mentioned as a way to allow more artistic control and easier balancing. There's also a suggestion to treat armor as another type of tool within this system or to have a separate crafting station for it. User comments introduce the idea of procedural armor with no distinction between armor and accessories, suggesting specific slots for different types of equipment.
