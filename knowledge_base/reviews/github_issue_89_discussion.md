## Summary
Discussion on procedural crafting system, focusing on separating tool attributes into 'Power' and 'Hardness', implementing a dropdown for tool type selection, and considering procedural armor.

## Explanation
Discussion on refining the procedural crafting system in Cubyz. The maintainers propose separating tool attributes into 'Power' and 'Hardness': Power determines how much damage a tool does to blocks, while Hardness is based on the Mohs Scale. A dropdown menu during tool creation allows selecting specific tool types (e.g., pickaxe or sword), applying corresponding formulas for stats determination. The implementation of #594 aims to provide artistic control and easier balancing by simplifying the UI and underlying equations. Armor crafting is considered either as another type of tool within this system or via a separate crafting station. User comments suggest procedural armor with no distinction between armor and accessories, proposing specific slots like ring, helmet, goggles, chest, arms, legs, etc.
