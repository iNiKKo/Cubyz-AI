## Summary
Discussion about expanding Cubyz's inventory size from 32 slots to 48 slots by adding an extra row and increasing row width to 12 slots.

## Explanation
The issue discusses expanding Cubyz's inventory size, which currently has 32 slots (smaller than Minecraft's 36 slots). The proposal is to increase it to 48 slots by adding an extra row and making each row 12 slots wide. This design would be similar to the hotbar in other games. There are discussions about implementing additional features like #73 and #1060 before balancing the inventory slot count. Users also suggest using the hotbar for scrolling through different inventories and question whether inventory management should be a player concern given the introduction of bags.
