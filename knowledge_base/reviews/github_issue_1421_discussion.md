## Summary
Discussion about adding a dedicated help window for displaying all available commands.

## Explanation
Discussion about adding a dedicated help window for displaying all available commands. The issue proposes that this custom window would improve quality of life (QOL) by allowing players to view commands without scrolling through chat messages. However, the maintainer suggests implementing this as a separate button in the UI rather than altering command handling, which could disrupt functionality on headless servers where commands are executed without a graphical interface.
