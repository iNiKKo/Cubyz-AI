## Summary
The proposal to add magnesium to the game has been revised after concerns about its magnetic property potentially negatively impacting younger players. Magnesium is now proposed as a source for brighter torches, similar to current white crystals.

## Explanation
The original proposal included making magnesium magnetic, which was later abandoned due to potential negative effects on younger players during gameplay. The maintainers decided that magnesium would be more suitable as a resource for creating brighter torches, aligning it with the game's progression towards end-game items. This change aims to balance the game's content and ensure accessibility for all age groups.
