## Summary
Discussion on improving user feedback for chests that are still loading by suggesting visual indicators like darkened slots or a 'loading' texture.

## Explanation
The issue revolves around the lack of clear visual feedback to users when a chest's contents are still being loaded, potentially leading to confusion. The discussion proposes two solutions: rendering the chest's slots darker until they load (similar to empty slots in the creative menu) or using a 'loading' texture to indicate that the chest is still processing. These suggestions aim to enhance user experience by providing immediate visual cues about the loading status of chests.
