## Summary
Discussion about implementing tool tags in Cubyz, focusing on user interface design for displaying tag information.

## Explanation
The discussion revolves around the implementation of tool tags in Cubyz, which allows specific modifiers to be activated based on materials placed in certain slots during crafting. The maintainers are seeking input on how these tags should be visually represented to users. One suggestion is using icons similar to Minecraft's armor slots, with additional functionality to display tag names and stats upon hovering over the icon. This approach aims to enhance user experience by providing clear and interactive feedback about the modifiers applied to tools.
