## Summary
Discussion about restructuring menu options for advanced controls in Cubyz.

## Explanation
The discussion revolves around organizing the user interface of Cubyz, specifically regarding the placement and naming of advanced control settings. The maintainer initially decided to keep 'Advanced Controls' as a separate item under the main 'Controls' menu to avoid burying power-user settings deeply within nested menus. However, there is some dissatisfaction with the current naming convention, particularly for 'Advanced Controls'. A user suggests renaming 'Controls' to 'Keybinds' and 'Advanced Controls' to simply 'Controls', while also proposing to move mouse sensitivity settings into this new 'Controls' menu. This proposal aims to streamline the interface and potentially make it more intuitive for users.
