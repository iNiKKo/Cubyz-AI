## Explanation
The discussion revolves around adding keyboard input for setting exact values in the settings menu, with concerns about player perception of small differences and potential UI refinement. Currently, players can only adjust settings like FPS limit, FOV, mouse sensitivity, and others using sliders, making it difficult to set exact values. A user suggests implementing text input fields next to the sliders to allow direct entry of specific values. However, the maintainer argues that minor differences (e.g., between 70 and 71 degrees) are unlikely to be noticeable for most users and proposes displaying coarser resolution values instead. This debate highlights considerations in user experience and interface design.
