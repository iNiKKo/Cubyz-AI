## Summary

## Explanation
The issue highlights a usability problem where players cannot set precise values directly through the settings menu, which currently only offers sliders. The user proposes adding text input fields next to the sliders to address this. However, the maintainer argues that minor differences in values (e.g., between 70 and 71 degrees) are unlikely to be noticeable and suggests displaying coarser resolution values instead. This debate touches on user experience and interface design considerations.
