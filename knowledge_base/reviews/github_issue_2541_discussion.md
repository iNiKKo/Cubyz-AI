## Summary
Discussion on implementing an identification mechanic where loot can be unidentified, requiring a Scroll of Identification or similar method to reveal its stats. Various methods for identifying items are proposed, including workstations and player interaction with the environment.

## Explanation
The discussion revolves around the implementation of an identification mechanic in Cubyz, where most loot found in chests is initially unidentified, meaning players cannot see their stats. The primary method proposed is using a Scroll of Identification to reveal these stats. However, there are alternative suggestions such as using workstations for examination and research, or having players interact with the environment to learn item properties. The maintainer expresses preference for more tangible methods like interacting with blocks or using items themselves to gain knowledge. There's also a suggestion that identifying materials could provide additional stats like weight and hardness.
