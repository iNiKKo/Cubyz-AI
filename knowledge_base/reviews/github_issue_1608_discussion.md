## Summary
Discussion about procedural crafting stations needed for various items like armor and weapons.

## Explanation
The discussion revolves around identifying the necessary procedural crafting stations in Cubyz to support the creation of different types of items such as armor, weapons, and accessories. The maintainer suggests considering traditional tools like anvils for metal-based crafts (including weapons) and potentially adding an Anvil Horn for crafting jewelry. For armor, it is suggested that an anvil might be used or a Tool Rack could be added to allow the creation of specialized items such as saws and cutters. Armor is traditionally made on an anvil using metal sheets. This issue aims to ensure that the game's crafting system is comprehensive and supports a wide range of item creation processes.
