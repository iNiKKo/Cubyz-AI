## Summary
The server join message is outdated as players can now join servers without providing their IP address.

## Explanation
The issue highlights an outdated user interface element in the Cubyz game client. The current server join message prompts users to provide their IP address, which is no longer necessary due to changes in how servers are joined. The maintainer suggests implementing two separate screens: one for joining public servers and another for playing with friends. This change aims to improve the user experience by providing more relevant information and options based on the context of server joining.
