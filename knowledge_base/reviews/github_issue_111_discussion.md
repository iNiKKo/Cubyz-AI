## Summary
The maintainer has decided against adding procedurally generated ores to the Zig version of Cubyz, opting instead for handcrafted and hand-textured ores.

## Explanation
The issue report highlights that the procedural ore generation in the Java version of Cubyz is considered boring and lacks purpose. The reviewer reflects on the systems used previously, noting that while they weren't entirely bad, the name generator could be useful for inspiration and the texture generator was actually utilized for current ore textures. However, the maintainer has made a decision to not include procedural generation in the Zig version, believing that handcrafted ores are more useful and interesting.
