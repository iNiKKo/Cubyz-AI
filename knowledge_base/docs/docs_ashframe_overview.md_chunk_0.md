## Summary
Ashframe is iNiKKo's personal brand for game servers and community projects, separate from Cubyz. It operates dedicated servers for Satisfactory, Minecraft, and Cubyz, as well as a website for sharing addons.

## Explanation
Ashframe is iNiKKo's umbrella for game servers and related community projects, distinct from the Cubyz project itself. iNiKKo also created the Cubyz AI assistant. Ashframe operates dedicated servers for Satisfactory (no longer active), Minecraft (no longer active), and Cubyz (currently active). Additionally, it hosts addons.ashframe.net, known as 'Cubyz Hub,' a website where players can upload, share, and download addons for Cubyz.
