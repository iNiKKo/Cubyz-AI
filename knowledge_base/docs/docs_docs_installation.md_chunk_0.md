## Summary
Cubyz can be installed either by downloading a pre-built release or compiling from source. The minimum hardware requirements include Windows/Linux OS, specific GPUs, dual-core CPUs, and at least 4 GB of RAM.

## Explanation
To play Cubyz, users have two main installation options: downloading a ready-to-run release from the GitHub releases page or compiling the latest development version themselves using source code. Minimum hardware requirements: OS is Windows or Linux; example GPUs meeting spec are Radeon Vega 8, Intel HD Graphics 530, or NVIDIA GTX 750 -- the GPU must have hardware support for OpenGL 4.6 specifically, with up-to-date drivers; example CPUs are a dual-core 64-bit processor such as Ryzen 3 2200G or Intel i3-6100; RAM requirement is at least 4 GB. For downloading, users extract the archive containing `Cubyz`, `launchConfig.zon`, and `assets` files, then run the executable. Compiling from source can be done either by downloading a ZIP file and running a launch script (`run_linux.sh` on Linux, `run_windows.bat` on Windows) or using Git to clone the repository and compile it directly.
