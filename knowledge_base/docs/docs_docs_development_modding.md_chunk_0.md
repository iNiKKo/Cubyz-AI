## Summary
The official compile-time modding API is incomplete; progress tracked in issue #1528 with initial discussions in #1507. Unofficial forks offering custom modding support include a Web Assembly addon, an ElevatorMod repository, and a Lua integration branch.

## Explanation
Cubyz currently lacks a finished official compile-time modding API; users can monitor its development via GitHub issue #1528 (progress) and #1507 (initial discussions). For modding needs before the official API is complete, three unofficial options are documented: Web Assembly modding available at addons.ashframe.net (addon ID 24), a compile-time modding fork hosted on GitHub under KewaiiGamer/ElevatorMod-Cubyz, and a Lua integration fork maintained by LinaPlusPlus/Cubyz in the integrate_lua branch.
