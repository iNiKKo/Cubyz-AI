## Summary
Cubyz supports two crafting methods: a Crafting Menu that auto-displays all craftable items, and a Workbench UI where any material item can be combined to produce fast/damaging or durable/slow outputs by selecting the target via a top-right button.

## Explanation
The game provides a Crafting Menu which automatically lists every possible craftable item for the player. Alternatively, players may use a Workbench interface; any block or item tagged as a material can be placed into it to combine with other materials. The resulting crafted item's properties (speed and damage) depend on the combination chosen, allowing creation of either high-damage/fast items or durable/slow items. Selection of which recipe to execute is performed by pressing the button located at the top right corner of the Workbench UI.
