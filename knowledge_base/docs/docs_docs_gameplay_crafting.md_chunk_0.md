## Summary
Cubyz offers two crafting methods: via a Workbench and via the Crafting Menu. The Crafting Menu automatically displays all craftable items, while the Workbench allows combining materials to create specific items.

## Explanation
In Cubyz, players have two primary options for crafting: using the Crafting Menu or utilizing a Workbench. The Crafting Menu is an automated interface that lists all possible items that can be crafted based on available resources. This menu simplifies the process by eliminating the need to manually search through recipes.

The Workbench provides a more interactive and customizable crafting experience: it is the art of combining various items to see what you make. **Any item considered a `material` can be used for Workbench crafting** -- that `material` tag is the sole qualifier, not any other item property. Players can combine such materials to create fast/highly-damaging items or durable/slow items. The Workbench UI includes a button on the top right that enables players to select which item they wish to craft from the available materials.
