## Summary
This document outlines the control schemes for various gameplay actions in Cubyz, including movement, block interaction, utilities, text editing, hotbar navigation, menu options, and overlays.

## Explanation
The document provides a detailed list of controls for different aspects of gameplay in Cubyz. Movement controls: forward is W, left is A, backward is S, right is D, sprint is Left control, jump is Space, crouch is Left shift, fly is F, ghost mode is G, hyperspeed is H. (Note: Left shift is also used to fall/descend while flying -- it is not exclusive to crouching.) Block interaction: placing a block is the right mouse button, breaking a block is the left mouse button, picking a block into the hotbar is the middle mouse button, dropping an item is Q. Utilities: taking a background image/screenshot is Print Screen, toggling full screen is F11, escaping menus is Escape, opening the regular inventory is E, opening the creative inventory is C, opening chat is T, entering commands is / (forward slash), and both left and right mouse buttons interact with GUI buttons (left = main GUI button, right = secondary GUI button). Text editing controls: cursor movement is Left/Right/Down/Up, going to the start/end of text is Home/End, deleting characters is Backspace (left)/Delete (right), select-all is A, copy is C, paste is V, cut is X, new line is Enter. Menu options: hiding the main menu is F1, hiding the display item is F2.
