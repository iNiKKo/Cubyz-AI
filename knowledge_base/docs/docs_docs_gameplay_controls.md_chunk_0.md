## Summary
Cubyz's movement controls: one key per movement action, stated individually.

## Explanation
The key that moves the player forward is W.
The key that strafes/moves the player left is A.
The key that moves the player backward is S.
The key that strafes/moves the player right is D.
The key that makes the player sprint is Left control.
The key that makes the player jump is Space.
The key that makes the player crouch is Left shift. Left shift is also used to fall/descend while flying -- it is not exclusive to crouching.
The key that toggles flying is F.
The key that toggles ghost mode is G.
The key that activates hyperspeed is H.
