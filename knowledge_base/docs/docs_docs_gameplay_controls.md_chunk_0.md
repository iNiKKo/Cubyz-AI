## Summary
This document outlines the control schemes for various gameplay actions, block interactions, utilities, text editing, hotbar navigation, menu operations, and overlay toggles in Cubyz.

## Explanation
Movement: Forward is W, Left is A, Backward is S, Right is D, Sprint is Left control, Jump is Space, Crouch is Left shift, Fly is F, Ghost is G, Hyperspeed is H, and Fall (fly down while flying) is also Left shift. Blocks: Place is Right mouse, Break is Left mouse, Pick block is Middle mouse, Drop is Q. Utilities: TakeBackgroundImage is Print Screen, FullScreen is F11, Escape is Escape, Inventory is E, Creative inventory is C, Chat is T, Command is /, MainGuiButton is Left mouse, SecondaryGuiButton is Right mouse. Text editing: TextCursorLeft is Left arrow, TextCursorRight is Right arrow, TextCursorDown is Down arrow, TextCursorUp is Up arrow, TextGotoStart is Home, TextGotoEnd is End, TextDeleteLeft is Backspace, TextDeleteRight is Delete, TextSelectAll is A, TextCopy is C, TextPaste is V, TextCut is X, TextNewLine is Enter. Hotbar slots 1 through 9 use keys 1-9, slot 10 uses 0, slot 11 uses -, slot 12 uses =. Menu: HideMenu is F1, HideDisplayItem is F2. Overlays: DebugOverlay is F3, PerformanceOverlay is F4, GpuOverlay is F5, NetworkOverlay is F6, AdvancedNetOverlay is F7.
