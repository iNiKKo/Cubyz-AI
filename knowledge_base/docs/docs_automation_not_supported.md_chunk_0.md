## Explanation
In Cubyz, there are no conveyor belts, auto-crafters, factories, or redstone-like circuitry. All crafting must be done manually using either the Crafting Menu, which lists all possible craftable items, or the Workbench, where materials are combined by hand to produce items with varying properties such as speed/damage or durability/slow properties. There is currently no automation for resource gathering, item transport, or crafting.
