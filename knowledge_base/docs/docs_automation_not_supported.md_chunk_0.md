## Explanation
Cubyz does not support machines or automation features such as conveyor belts, auto-crafters, factories, or redstone-like circuitry. Players must perform all crafting tasks manually using either the Crafting Menu for automatic item listing or the Workbench for combining materials by hand to create items with specific properties like speed/damage or durability/slow.
