## Explanation
Cubyz lacks any machine or automation systems, including conveyor belts, auto-crafters, factories, or redstone-like circuitry. All crafting must be done manually using either the Crafting Menu for automatic item listing or the Workbench for combining materials by hand to create items with specific properties like speed/damage or durability/slow. There is currently no way to automate resource gathering, item transport, or crafting in Cubyz.
