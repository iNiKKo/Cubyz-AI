## Summary
This section covers third-party services and add-ons in Cubyz, including downloading and installing add-ons, setting up a Discord bot named Mercur, and configuring the Cubyz Server List.

## Explanation
Add-ons are community-made extensions that modify or introduce new content to Cubyz. They can be downloaded from the official Discord server or an online marketplace. To install add-ons, users need to navigate to their world save directory, extract the archive, and place it in the `assets` folder. The Mercur Discord bot facilitates communication between Discord and Cubyz servers and can submit server information to the Cubyz Server List. The installation of Mercur requires Node.js and npm. Configuration involves setting up a `config.json` file with necessary details such as server name, IP address, port, description, icon URL, and Discord invite link. Updates to the bot can be done by running the latest version command and handling configuration changes if necessary.
