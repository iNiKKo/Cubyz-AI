## Summary
Cubyz's balancing principles: the "2OP4ME" rule and trade-offs. **From GAME_DESIGN_PRINCIPLES.md, a contributor-facing design-vision document -- the "enemies" mentioned in the trade-offs example is a design goal, not confirmation that enemy combat is currently implemented.**

## Explanation
When balancing the game, keep in mind how players might interact with the world, the wildlife, and each other.

2OP4ME: players need to be vulnerable at all times to avoid power imbalances. Armor, tools, accessories, and buffs should aid the player, not let them win outright.

Trade-offs: if the player is given something to aid them, it should have an appropriate take-away to balance it. Examples: using rare resources to create a strong tool; an accessory that heals the player but takes away energy or some other resource; enemies having strengths and weaknesses towards particular damage types.
