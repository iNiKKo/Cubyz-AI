## Summary
This page provides guidance on creating addons for Cubyz, detailing the folder structure and file types required.

## Explanation
The guide explains that all addon content should be placed within a folder named after the addon. Depending on the type of content to be added, various subfolders such as Biomes, Blocks, Items, Models, Particles, Recipes, and SBB (Structure Building Blocks) may need to be created inside this main folder. Most data is stored in `zig.zon` files, where each field is prefixed with a dot (`.`).
